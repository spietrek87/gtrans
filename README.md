## Description
Copies SAP® co- and data files and generates a ZIP file.

## Usage
```
gtrans.py <directory> <transport_request> "<description>"
```

## Example
```
python gtrans.py \\NETWORKDIR\trans SY1K900123 "zip_file_name_without_extension"
```
